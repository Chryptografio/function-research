Нули функции

Нули косинуса на интервале [-2π,2π] (красные точки)

Нуль функции — элемент из области определения функции, в котором функция принимает нулевое значение. Например, для функции f, заданной формулой f(x)=x^2-6x+9, x = 3 является нулём, поскольку f(3) = 32-6*3+9=0.

Интервалы знакопостоянства

Интервал знакопостоянства – это интервал, в каждой точке которого функция положительна либо отрицательна. В нашем случае функция положительна на интервалах (a;b), (c;d), (d;e), то есть для любого значения «икс» любого из перечисленных интервалов справедливо строгое неравенство f(x)>0. Или совсем просто – график функции на таких интервалах расположен ВЫШЕ оси абсцисс.
