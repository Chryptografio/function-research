Полное исследование функции и
построение ее графика

Исследование функций является неотъемлемой частью математического анализа. Под полным исследованием функции подразумевается прохождение по всему плану, представленному ниже.

План исследования функции:

  1. Область определения функции
  2. Нули функции, интервалы знакопостоянства
  3. Четность
  4. Непрерывность, дифферинцируемость
  5. Асимптоты
  6. Критические точки, монотонность
  7. Точки локальных экстремумов, локальные экстремумы
  8. Вторая производная, выпуклость, точки перегиба
  9. Область значений функции

Построение графика

Завершающим этапом исследования функции является построение ее графика. Для этого полученные данные заносятся в сводную таблицу, что позволяет не ошибиться при построении. Ниже приведен пример сводной таблицы для функции f:D→R, f(x)=ln(x2-4). Т.к. эта функция - четная (график симметричен относительно оси ординат), можно сделать сводную таблицу только для левой/правой полуплоскости.
